# GlucoCompanion

Eine Diabetes Self-Management App entwickelt mit Next.js 14, TypeScript und Tailwind CSS.

## Setup

1. Installiere die Abhängigkeiten:
```bash
npm install
```

2. Starte den Development Server:
```bash
npm run dev
```

3. Öffne [http://localhost:3000](http://localhost:3000) im Browser.

## Projektstruktur

- `src/app/` - Next.js App Router Seiten und Layouts
- `src/components/` - React Komponenten
  - `ui/` - shadcn/ui Komponenten
  - `shared/` - Geteilte Komponenten
  - `dashboard/` - Dashboard Komponenten
  - `logbook/` - Tagebuch Komponenten
  - `companion/` - Begleiter Komponenten
  - `insights/` - Einblicke Komponenten
  - `import/` - Import Komponenten
- `src/hooks/` - Custom React Hooks
- `src/i18n/` - Internationalisierungs-Dateien
- `src/lib/` - Utility-Funktionen und Konstanten

## Technologien

- Next.js 14 mit App Router
- TypeScript
- Tailwind CSS
- shadcn/ui
- next-intl (für Internationalisierung)
- lucide-react (Icons)
- recharts (Charts)

## Features

- Mehrsprachigkeit (Deutsch/Englisch)
- Responsive Design
- Moderne UI-Komponenten
